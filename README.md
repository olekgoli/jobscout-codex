# Job Scout for Codex

Publiczny pakiet instalacyjny Job Scout. Zawiera wyłącznie skill, konfigurację
read-only MCP i instalator Codexa. Nie zawiera kodu aplikacji ani klucza MCP.

## Instalacja

Wklej ten link do lokalnego zadania w Codexie:

```text
https://github.com/olekgoli/jobscout-codex
```

Napisz:

```text
Zainstaluj. Sprawdź i uruchom install_codex.py. Nie proś mnie o klucz w czacie;
instalator ma zebrać go w ukrytym lokalnym oknie.
```

Codex pobierze i sprawdzi `install_codex.py`, a następnie go uruchomi. Instalator:

1. poprosi o klucz w ukrytym lokalnym oknie;
2. sprawdzi połączenie z Job Scout;
3. zainstaluje plugin, skill i MCP;
4. zapisze konfigurację Codexa z uprawnieniami `0600`.

Po instalacji uruchom Codex ponownie, rozpocznij nowe zadanie i napisz
`znajdź mi pracę`.

Klucz przekaż przez menedżer haseł lub inny prywatny kanał. Nie wklejaj go do
czatu, zgłoszenia ani zrzutu ekranu.
